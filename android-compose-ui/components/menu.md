# Menu

## Use when

- A control should reveal a temporary anchored list of secondary choices or actions.

## Core APIs

- `DropdownMenu`
- `DropdownMenuItem`

## Guidance

- Often triggered by an `IconButton`.
- Good for overflow or contextual secondary actions.
- Avoid hiding clear primary actions in a menu unless space requires it.

- https://developer.android.com/develop/ui/compose/components/menu
