# Navigation Drawer

## Use when

- The app has broader navigation density, account/context links, or a larger-screen side-menu pattern.

## Variants

- `ModalNavigationDrawer`
- Drawer content typically inside `ModalDrawerSheet`

## Guidance

- Use when bottom bar or rail is not enough.
- Good for complex products or large-screen layouts.

## Key APIs

- `drawerContent`
- `NavigationDrawerItem`
- `gesturesEnabled`

## Official source

- https://developer.android.com/develop/ui/compose/components/drawer
