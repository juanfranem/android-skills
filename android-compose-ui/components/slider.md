# Slider

## Use when

- The user selects a value or range inside bounded limits and approximate manipulation is acceptable.

## Variants

- `Slider`
- `RangeSlider`
- Continuous or discrete via `steps`

## Guidance

- Use for tunable numeric values and interval filtering.
- If precision typing matters more than touch manipulation, a text field may be better.

## Key APIs

- `valueRange`
- `steps`
- `colors`

## Official source

- https://developer.android.com/develop/ui/compose/components/slider
