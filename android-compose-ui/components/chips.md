# Chips

## Use when

- The UI needs compact action, filter, suggestion, or chosen-entity tokens.

## Variants

- `AssistChip`
- `FilterChip`
- `InputChip`
- `SuggestionChip`

## Guidance

- Use assist chips for lightweight contextual actions.
- Use filter chips for on/off filters.
- Use input chips for chosen entities with optional removal.
- Use suggestion chips for recommended next actions.

## Nearby alternatives

- Larger or more structured selection set -> segmented buttons, menus, or checkboxes depending on context

- https://developer.android.com/develop/ui/compose/components/chip
