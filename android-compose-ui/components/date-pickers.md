# Date Pickers

## Use when

- The user needs structured date or date-range input.

## Variants

- `DatePicker`
- `DatePickerDialog`
- `DateRangePicker`

## Guidance

- Choose docked or modal presentation based on the surrounding flow.
- Use range picker for explicit start/end selection.

## Important note

- These APIs are documented as experimental on the official pages. Mention `@OptIn(ExperimentalMaterial3Api::class)` when relevant.

## Official source

- https://developer.android.com/develop/ui/compose/components/datepickers
