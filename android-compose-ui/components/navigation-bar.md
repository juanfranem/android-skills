# Navigation Bar

## Use when

- The app has a small set of top-level destinations on compact layouts.

## Core APIs

- `NavigationBar`
- `NavigationBarItem`

## Guidance

- Best for around 3 to 5 equally important destinations.
- Hoist selected destination state.

## Nearby alternatives

- Medium/larger side navigation -> `NavigationRail`
- Broader feature space -> drawer

## Official source

- https://developer.android.com/develop/ui/compose/components/navigation-bar
