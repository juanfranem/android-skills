# Navigation Rail

## Use when

- The layout is medium or large enough that side navigation is more ergonomic than bottom navigation.

## Core APIs

- `NavigationRail`
- `NavigationRailItem`

## Guidance

- Good for tablets, desktop-like layouts, and wider phones in landscape.
- Works well with a manageable set of top-level destinations.

## Nearby alternatives

- Compact layout -> `NavigationBar`
- Much broader navigation set -> drawer
