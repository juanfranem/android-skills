# Divider

## Use when

- Spacing alone is not enough to separate groups visually.

## Variants

- `HorizontalDivider`
- `VerticalDivider`

## Guidance

- Use sparingly.
- Prefer layout hierarchy and spacing first.
- Drive spacing through modifier usage around the divider.

## Official source

- https://developer.android.com/develop/ui/compose/components/divider
