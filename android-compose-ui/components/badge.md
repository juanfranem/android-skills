# Badge

## Use when

- You need to attach a count, status, or freshness indicator to another UI element.

## Core composables

- `Badge`
- `BadgedBox`

## Guidance

- Use for unread counts, task counts, or simple new/status indicators.
- Keep content short.

## Nearby alternatives

- Richer status explanation -> snackbar, supporting text, or chip depending on context

## Common mistakes

- Putting long text inside a badge.
- Using badge as a standalone primary status surface.

## Official source

- https://developer.android.com/develop/ui/compose/components/badges
