# Card

## Use when

- You need a semantic container for one coherent subject such as a product, article, message, or settings group.

## Variants

- `Card`
- `ElevatedCard`
- `OutlinedCard`

## Guidance

- `Card` is the neutral default.
- `ElevatedCard` adds prominence.
- `OutlinedCard` separates without stronger elevation.
- Use cards when grouping improves meaning or scanability, not as a default wrapper for everything.

## Key APIs

- `CardDefaults.cardColors()`
- `CardDefaults.cardElevation()`
