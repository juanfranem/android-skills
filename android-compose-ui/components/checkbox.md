# Checkbox

## Use when

- The user can select multiple independent items, or when a parent-child selection model needs indeterminate state.

## Variants

- `Checkbox`
- `TriStateCheckbox`

## Guidance

- Use tri-state when an aggregate parent reflects mixed child state.
- If the label reads like an immediate setting toggle, consider `Switch` instead.
