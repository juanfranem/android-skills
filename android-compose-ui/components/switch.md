# Switch

## Use when

- The control reads as an immediate on/off setting.

## Core APIs

- `checked`
- `onCheckedChange`
- `thumbContent`
- `colors`

## Guidance

- Prefer for settings and enable/disable states.
- If the control is really item selection rather than a setting, prefer checkbox.

- https://developer.android.com/develop/ui/compose/components/switch
