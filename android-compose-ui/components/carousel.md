# Carousel

## Use when

- Content is visual and browsable, and partial adjacent items help discovery.

## Variants

- `HorizontalMultiBrowseCarousel`
- `HorizontalUncontainedCarousel`
- Design guidance also references hero and full-screen styles

## Guidance

- Good for media-forward collections.
- Avoid when users need dense scanning or precise comparison.

## Key APIs

- `CarouselState`
- `rememberCarouselState()`
- `itemSpacing`
- `contentPadding`

## Official source

- https://developer.android.com/develop/ui/compose/components/carousel
