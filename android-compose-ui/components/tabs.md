# Tabs

## Use when

- The screen contains sibling content sections within the same feature.

## Variants

- `PrimaryTabRow`
- `SecondaryTabRow`
- `Tab`

## Guidance

- Primary tabs represent stronger hierarchy.
- Secondary tabs represent nested or subordinate partitions.
- If the control is just a local mode switch, consider segmented buttons instead.
