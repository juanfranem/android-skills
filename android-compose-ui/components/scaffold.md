# Scaffold

## Use when

- A screen needs coordinated Material layout regions such as top bar, bottom bar, FAB, or snackbar host.

## Core slots

- `topBar`
- `bottomBar`
- `floatingActionButton`
- `snackbarHost`
- content lambda with inner padding

## Guidance

- Treat `Scaffold` as screen structure, not a style wrapper.
- Apply the provided padding values to the screen content.

## Nearby alternatives

- Very simple surfaces may not need `Scaffold`.

- https://developer.android.com/develop/ui/compose/components/scaffold
